# Mini-Project-Crop-Yield-Prediction
A data analytics project for predicting crop yield using environmental and soil parameters with data visualization and Streamlit UI.
# 🌱 Smart Agriculture Analytics for Crop Yield Prediction

## 📌 Project Description

This project focuses on predicting crop yield using machine learning techniques. It analyzes important environmental and soil parameters such as rainfall, temperature, humidity, nitrogen, phosphorus, and potassium to estimate crop production. The system is implemented using Python and includes data analysis, visualization, and a Streamlit-based web application for user interaction.

---

## 🚀 Features

* Crop yield prediction using Machine Learning
* Data preprocessing and analysis
* Visualization using graphs and heatmaps
* Interactive user interface using Streamlit
* Easy input system for prediction

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## ▶️ How to Run the Project

### Step 1: Clone the Repository

```
git clone https://github.com/jagannathan1651/Mini-Project-Crop-Yield-Prediction
.git
cd Mini-Project-Crop-Yield-Prediction

```

### Step 2: Install Required Libraries

```
pip install pandas numpy matplotlib seaborn scikit-learn streamlit
```

### Step 3: Run the Application

```
python -m streamlit run app.py
```

### Step 4: Open in Browser

```
http://localhost:8501
```
